# Kitchen-mistery

## Descripción
Es un videojuego de gestión y sigilo en 3D donde el jugador administra un restaurante y debe conseguir recursos de forma clandestina sin ser descubierto. El objetivo es mejorar el local, optimizar la producción y mantener el negocio funcionando mientras se evita levantar sospechas.

## Género
Gestión, Sigilo, Estrategia, Simulación 3D

## Plataforma objetivo
PC

## Tecnologías
- Unity 3D
- C#
- GitHub para control de versiones

1. Abrir el proyecto en Unity Hub.
2. Cargar la escena principal.
3. Presionar Play para ejecutar el prototipo.

## Flujo de trabajo
El proyecto utiliza un flujo de trabajo basado en Git Flow simplificado:
- `main`: versión estable
- `develop`: integración de avances
- `feature/*`: desarrollo de nuevas funciones
